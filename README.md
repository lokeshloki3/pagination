## **Key Features**
* **Data Fetching with Pagination**:
  * Implemented efficient data fetching from an API or data source using pagination.
  * Loads a limited number of records per page to optimize performance and user experience.
* **Pagination Controls**:
  * Integrated **Next** and **Previous** buttons to navigate between pages.
  * Displayed the current page number and total pages to keep users informed.
* **Dynamic UI Rendering**:
  * Rendered paginated data in real-time using components like **images and titles**.
  * Automatically updated the view based on the selected page.
* **Loading & Error Handling**:
  * Gracefully handled errors and displayed fallback messages when needed.
* **State Management**:
  * Maintained current page and fetched data using local or global state management.
  * Enabled smooth transitions without losing user context.
